# Cypress Automation: Practice Test Login

Automated end-to-end tests for validating login functionality on [Practice Test Automation](https://practicetestautomation.com/practice-test-login/) using Cypress.

# Project technologies

- JavaScript as programming language

- Node.js as runtime environment

- Cypress as automation framework for web applications

- Google Chrome as test browser

# Local Set Up
```bash
1. Clone the repository to your local computer

git clone https://github.com/juniorcaba/Practice-Test-Automation.git

2. Install dependencies
npm install

3. Run tests
npx cypress run

4. Cypress Test Runner (GUI)
npx cypress open
```
